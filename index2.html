The way you type the name is EXACTLY how it will show on the jersey. For example, the three names below will show differently, based on how you capitalize the letters:

  • McKINLEY
  • McKinley
  • mckinley

In the example above if you type it all in lowercase it will be made in lower case! We will not change it to what you “wanted”, but will make it exactly how you type it.

SAVE OPTIONS

Clicking this will send the order to your locker where a Customer Service Rep can finalize the order and submit it to the factory. If you are working on something and would like to finish it later or you like the design so much you want to keep it for later then use this button.
*If you save this mock-up to a locker it will NOT be submitted as an order yet. That will take place after someone contacts you to see if you’d like to finalize the process. Don’t worry about this going to the factory and being made, that isn’t until later.
